## Benchmark 2 — Vue.js vs React vs Angular

### Problem addressed
Datalyz requires a frontend stack that is fast to develop, component-based, readable, reusable, and easy to integrate into both the browser extension and the dashboard.

### Compared solutions
- Vue.js
- React
- Angular

### Evaluation criteria
- Development speed
- Code readability
- Component model
- State management simplicity
- Ecosystem maturity
- Bundle/performance profile
- Compatibility with the extension stack

### Findings
Our comparison highlighted that frontend framework choice directly affects development speed, readability, code reuse, and long-term maintainability. The benchmark material also included ecosystem and performance comparisons between Vue, React and Angular.

### Decision
Vue.js was retained because it offered the best balance for Datalyz between development simplicity, modularity, integration with WXT, and maintainable frontend architecture.

### Impact on the project
This choice made it easier to standardize the popup, dashboard, routing, state handling, and internationalization around one coherent frontend stack.
